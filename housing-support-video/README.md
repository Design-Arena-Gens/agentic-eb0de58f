## राष्ट्रीय आवास सहायता वीडियो गाइड

यह प्रोजेक्ट एक Next.js आधारित माइक्रोसाइट प्रदान करता है जो राष्ट्रीय आवास सहायता कार्यक्रम के लिए 60 सेकंड का शांत, भरोसेमंद हिंदी वीडियो प्रदर्शित करती है। वीडियो में उद्देश्य, पात्रता, ऑनलाइन आवेदन प्रक्रिया और मुख्य लाभ साफ़ आवाज़, साधारण दृश्य तथा हल्के बैकग्राउंड संगीत के साथ दिए गए हैं।

### वीडियो एसेट तैयार करना

वीडियो `public/videos/housing-support.mp4` में उपलब्ध है। यदि आपको इसे पुनः उत्पन्न करना हो:

1. आवश्यक पाइथन निर्भरताएँ स्थापित करें:
   ```bash
   python3 -m venv .venv
   . .venv/bin/activate
   pip install -r scripts/requirements.txt
   ```
2. वीडियो जेनरेशन स्क्रिप्ट चलाएँ:
   ```bash
   python scripts/generate_video.py
   ```
   स्क्रिप्ट स्वतः छवियाँ, हिंदी वॉयसओवर, बैकग्राउंड संगीत और अंतिम 60 सेकंड का वीडियो बनाती है।

### लोकल विकास

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) खोलें। मुख्य पृष्ठ पर वीडियो प्लेयर, प्रमुख बिंदु और उपयोग निर्देश हिंदी में उपलब्ध हैं।

### प्रोडक्शन बिल्ड

```bash
npm run build
npm run start
```

### परियोजना संरचना का सार

- `src/app/page.tsx` — लैंडिंग पेज जिसमें वीडियो प्लेयर और जानकारीपूर्ण सेक्शन हैं।
- `public/videos/housing-support.mp4` — उत्पन्न वीडियो फ़ाइल।
- `public/posters/housing-support.jpg` — वीडियो पोस्टर फ़्रेम।
- `scripts/generate_video.py` — एसेट्स और वीडियो बनाने के लिए स्क्रिप्ट।
- `scripts/requirements.txt` — स्क्रिप्ट के लिए आवश्यक पाइथन पैकेज सूची।
